Practice Ideas

Ghosted convergence checks

Draw a horizon and vanishing point, ghost each receding edge before committing, then extend the lines after the box to check how far they missed.

Rotated box belt

Start from one box and rotate the neighboring boxes step by step, keeping the gaps tight so each new edge borrows information from the one beside it.

Cylinder in a box

Lay in a box first, find the minor axis, fit both ellipses to that box, make the far ellipse open a little wider, and connect the side contours cleanly.

Rotation flipbook

Use the new rotate buttons to nudge one form a little at a time, and redraw each step on paper like a turntable study instead of jumping to a random new box.

Based on Drawabox and cylinder challenge notes.

Practice Ideas

5-minute warm-up

Spend five minutes on 30-second poses and hunt only for the line of action and the biggest weight shift, not anatomy or details.

1-minute torso map

In one minute, place the action line, head, ribcage, and pelvis first so the whole pose reads before you worry about arms, legs, or contour cleanup.

C-S-I simplification

Reduce the pose to C curves, S curves, and straights with as few lines as possible so the movement stays clear and the drawing does not stiffen up.

Story pass

On longer poses, exaggerate the action first and only then add structure, so the final figure keeps the motion that made the pose interesting.

Based on Proko and Line of Action.

Practice Ideas

Photo overlay check

Load a phone photo, place the horizon and vanishing points, then block the big forms with boxes and cylinders before tracing any details.

Same guides, new shapes

Keep one horizon setup and add several forms that all obey the same vanishing points. This is a good way to test whether your guide placement is actually consistent.
